# 📊 Task 04 - Sentiment Analysis

## 🎯 Objective
To analyze social media text data and classify sentiments as positive, negative, or neutral.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## 📌 Steps Performed
1. Loaded the dataset
2. Assigned column names and explored data
3. Cleaned text data and removed missing values
4. Visualized sentiment distribution
5. Converted text into numerical form using TF-IDF
6. Split data into training and testing sets
7. Built Naive Bayes model
8. Evaluated model using accuracy and confusion matrix
9. Visualized confusion matrix

## 💡 Key Insights
- Sentiment distribution shows variation across categories  
- Text preprocessing is important for NLP tasks  
- TF-IDF helps convert text into machine-readable format  
- Naive Bayes performs well for text classification  
- Model accuracy indicates effective sentiment prediction
